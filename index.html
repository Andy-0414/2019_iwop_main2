소개

오랜 역사를 자랑하고 있는소프트웨어과의 유일한 웹 개발 동아리, IWOP은 웹 개발과 웹 디자인 그리고 교내대회 기획에 대해 배울 수 있습니다.

커리큘럼

code

HTML

처음 IWOP에 들어오면 HTML5와 CSS3로 정적페이지를 만드는 법을 배웁니다.

style

CSS

HTML과 CSS를 배우고나서 기능적인 부분을 넣기 위해서 Node.js를 이용한 서버를 만드는 방법을 배우게 됩니다.

all_inbox

JS

MariaDB를 사용하여 데이터를 저장하여 활용하는 서버를 만드는 법을 배웁니다.
